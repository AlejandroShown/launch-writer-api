# Slop Dictionary — Kill on Sight

Every pattern below is banned. When found, replace with the suggested alternative or delete entirely.

## Corporate Slop
| Kill | Replace With |
|---|---|
| "We're thrilled to announce" | "Introducing" or bold claim |
| "Excited to share" | Just state it |
| "We're on a mission to" | State what you did |
| "Solutions for" | What it actually DOES |
| "Leverage" / "Utilize" | "Use" or specific verb |
| "Streamline" | Show the speed ("in 30 seconds") |
| "Optimize" (without specifics) | What specifically improves + by how much |
| "Best-in-class" | Prove it with a number |
| "World-class" | Prove it with a name |
| "State-of-the-art" | Name the technology |
| "Seamlessly" / "Effortlessly" | "In one click" / "In 30 seconds" |
| "Robust" / "Comprehensive" | Say what it covers specifically |
| "Empower" / "Enable" | Say what the user can now DO |
| "Ecosystem" / "Paradigm" / "Synergy" | Delete. Rewrite the sentence. |
| "Innovative" / "Revolutionary" | SHOW the innovation, don't label it |
| "Cutting-edge" | Name the specific technology |

## Filler Transitions
| Kill | Replace With |
|---|---|
| "Additionally" | Delete. Start new thought directly. |
| "Furthermore" | Delete. |
| "Moreover" | Delete. |
| "In addition" | Delete. |
| "It should be noted" | Delete. Just state it. |
| "It's worth mentioning" | Delete. Just say it. |
| "Importantly" | Delete. If it's important, it'll be obvious. |
| "Interestingly" | Delete. |
| "In conclusion" | Never. Just end with CTA. |
| "To summarize" | Never. |
| "That being said" | Delete. |
| "Moving on" | Use sacred pivot: "Now let's try something else." |

## Structural Slop
| Kill | Replace With |
|---|---|
| One-word sentences ("Done." "Period." "Boom.") | Combine with previous or delete |
| Three short fragments back to back | Rewrite as 1-2 proper sentences |
| "Not X. Not Y. Z." triple pattern | Rewrite as a single declarative statement |
| Bullet points in script body | Rewrite as flowing spoken sentences |
| Numbered lists ("Three reasons...") | Rewrite as narrative |
| Same word starting 2 consecutive sentences | Change one starter |
| Three sentences same length in a row | Vary: short → medium → short |

## Voice Slop
| Kill | Replace With |
|---|---|
| "I" (outside founder dialogue) | "You", "We", or product name |
| "Users" / "Customers" | "You" |
| "The platform" / "The tool" | Product's actual name |
| Passive voice ("is analyzed by") | Active: "[Product] analyzes" |
| Future tense ("will scan") | Present: "[Product] scans" |
| Hedging ("can help", "may improve") | Direct: "[Product] does X" |
| "Simple" / "Easy" | Show it: "In one click", "In 30 seconds" |

## Pitch Slop
| Kill | Replace With |
|---|---|
| "Have you ever...?" questions | Don't ask. State. |
| "Imagine if..." | Don't imagine. Show. |
| Explaining why AI is important | Viewer knows. Skip the preach. |
| Self-congratulation mid-body | Save for hook credibility line only |
| "We believe" / "We think" | State it as fact |
| Feature-first ("Our platform features...") | Narrate the feature in action |
